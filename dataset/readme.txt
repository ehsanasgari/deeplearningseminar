Toy dataset for deep learning seminar: a subset of hard set from https://peerj.com/articles/cs-90/ 

train_seq.txt: training sequences one sequence per line
train_label.txt: labels of sequences in train_seq.txt (1 for toxic 0 for non-toxic)

test_seq.txt: test sequences one sequence per line
test_label.txt: labels of sequences in test_seq.txt (1 for toxic 0 for non-toxic)
